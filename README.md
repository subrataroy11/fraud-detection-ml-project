# 💳 Fraud Detection using Machine Learning

## 📌 Project Overview
This project aims to detect fraudulent transactions using Machine Learning algorithms.  
The model analyzes transaction patterns and predicts whether a transaction is a fraud or not fraud.

---

## 🎯 Problem Statement
Online transactions are increasing rapidly, and fraud detection has become a critical issue.  
The goal of this project is to build a classification model that can accurately detect fraud transactions.
